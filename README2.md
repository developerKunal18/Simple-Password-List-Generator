# Simple Password List Generator

A Python script that generates common password combinations from a base word.

## Description
This project creates multiple password variations using a base word and common patterns.

## Features
- Generates multiple combinations
- Simple logic
- Useful for understanding patterns

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python password_generator.py

4. Enter a base word

## Example

Input:
admin

Output:
admin123  
admin@123  
admin2024  
admin_01  
Admin123  
admin!
